# Security Policy

## Reporting

Please report suspected security issues privately via GitHub Security Advisories or direct maintainer contact.

## Scope

This repository stores text and script assets for Copilot skill operations. Security-sensitive concerns include:

- malicious prompt/tool guidance
- unsafe shell/script patterns
- unintended privilege escalation guidance

## Response

- Acknowledge report within 3 business days.
- Triage and classify severity.
- Publish remediation summary after fix, when appropriate.
